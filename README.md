# 哈夫曼文件压缩器

基于哈夫曼编码的无损文件压缩工具，支持二进制文件的压缩与解压。

## 功能

- **统计频率**：读取文件，统计 256 种字节的出现频率
- **构建哈夫曼树**：使用优先队列（小根堆）构建最优前缀编码树
- **生成编码**：DFS 遍历树，为每个字符生成变长二进制编码
- **压缩**：将编码后的比特流打包成字节，写入压缩文件
- **解压**：读取频率表重建哈夫曼树，逐位解码还原原始文件
