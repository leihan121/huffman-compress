# 哈夫曼文件压缩器

基于哈夫曼编码的无损文件压缩工具，支持任意二进制文件。

---

## 功能

- **统计频率**：读取文件，统计 256 种字节出现次数
- **构建哈夫曼树**：优先队列（小根堆）构建最优前缀编码树
- **生成编码**：DFS 遍历，左"0"右"1"，为每个字符生成变长编码
- **压缩**：编码串每 8 位打包成 1 字节，不足补零
- **解压**：读取频率表重建哈夫曼树，逐位解码还原

---

## 六步通关

| 步骤 | 输入 | 输出 |
|:---|:---|:---|
| 1. 统计频率 | 原始文件 | freq[256] |
| 2. 构建哈夫曼树 | freq 数组 | 哈夫曼树根节点 |
| 3. 生成编码表 | 哈夫曼树 | a→0, b→10, c→11 |
| 4. 字符串编码 | 原始文件 + 编码表 | "010100..." |
| 5. 位级压缩 | 0101 字符串 | 二进制字节 |
| 6. 解码还原 | 压缩文件 | 原始文件 |

---

## 压缩文件格式
[原始大小 4B][频率表 1024B][压缩数据 ...]
