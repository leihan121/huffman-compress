# 哈夫曼压缩 · 六步通关

---

## 模块 1：统计频率（入门）

**目标**：读文件 → 统计 256 种字节各出现几次
输入：原始文件
输出：freq[256] 数组

| 字节 | 次数 |
|------|------|
| 'a' | 5 |
| 'b' | 3 |
| 'c' | 1 |

---

## 模块 2：构建哈夫曼树（算法核心）

**目标**：用频率 → 建树
小根堆取出两个最小节点 → 合并 → 塞回去
重复直到只剩一个根
    (9)
   /   \
 a(5)  (4)
      /   \
    b(3)  c(1)
    
---

## 模块 3：生成编码表

**目标**：DFS 遍历树 → 每个字符对应一串 0/1

| 字符 | 路径 | 编码 |
|------|------|------|
| a | 左 | `0` |
| b | 右→左 | `10` |
| c | 右→右 | `11` |

---

## 模块 4：字符串编码（过渡）

**目标**：把文件内容变成 0101 字符串

---

## 模块 3：生成编码表

**目标**：DFS 遍历树 → 每个字符对应一串 0/1

| 字符 | 路径 | 编码 |
|------|------|------|
| a | 左 | `0` |
| b | 右→左 | `10` |
| c | 右→右 | `11` |

---

## 模块 4：字符串编码（过渡）

**目标**：把文件内容变成 0101 字符串
原始：a b b a
编码：0 + 10 + 10 + 0 = "010100"

> 注意：此时还是字符串 '0' '1'，不是真正的二进制

---

## 模块 5：位级压缩（🔥 核心难点）

**目标**：把 "01010101" 变成 1 个字节
字符串 "01010001"
↓ 位运算
二进制 01010001 = 0x51 = 81
↓ 写入文件
1 字节

| 步骤 | 操作 |
|------|------|
| 缓冲 | `buffer += codes[ch]` |
| 攒够 8 位 | 截取前 8 位 |
| 位运算 | `byte <<= 1` + `byte \|= 1` |
| 写入 | `fout.write(&byte, 1)` |

---

## 模块 6：解码闭环

**目标**：压缩 → 解压 → 完全还原
读文件头 → 重建树 → 逐位遍历 → 输出字符

| 读取 | 方向 | 动作 |
|------|------|------|
| 0 | 左 | 走到 a，输出 |
| 1 | 右 | 继续 |
| 0 | 左 | 走到 b，输出 |

**停止条件**：输出够原始文件大小就停
